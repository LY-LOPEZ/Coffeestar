COFFESTAR - PHP + MYSQL + XAMPP

PASOS PARA INSTALAR:

1. Instala XAMPP.
2. Copia la carpeta "coffestar_php_mysql" en:
   C:\xampp\htdocs\

3. Abre XAMPP y activa:
   - Apache
   - MySQL

4. Entra a phpMyAdmin:
   http://localhost/phpmyadmin

5. Importa la base de datos:
   database/coffestar.sql

6. Abre el sistema:
   http://localhost/coffestar_php_mysql/
